[[Wine.Type]] is an enumeration that represents the **type** of a [[Wine]].
## Possible values
- `STILL` -- **Still** [[Wine]]s.
- `SPARKLING` -- **Sparkling** [[Wine]]s. For example, **Champagne**.
- `SPIRITOUS` -- **Spiritous Beverages**. For example, alcoholic tinctures.
